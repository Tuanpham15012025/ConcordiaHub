# 🎨 ConcordiaHub – UI Design Guide (Figma)

---

## 🏠 Home Page
- Project slogan: “In Concord We Evolve”
- Button: [Create Task]
- Recent Tasks list

---

## ➕ Create Task Page
- Input fields:
  - Task title
  - Task description
  - Task type: [Code / Content / Design]
  - Reward: [CORD] + Karma bonus
- Submit button triggers contract

---

## 📝 Review Task Page
- View task details
- Submit review text
- Task owner can:
  - Approve with rating
  - Reject with reason
- Reviewers see karma & rewards status

---

## 🎨 Style Guide
- **Fonts**: Inter, DM Sans
- **Color Scheme**: Earthy gold-brown (inspired by bees & symbiosis)
- **UI References**: StackOverflow + Web3 vibe
- **Button Style**: Rounded, soft shadow, responsive hover

---

## 🧠 UX Philosophy
- Promote contribution and clarity
- Fast review → fast reward
- Highlight user karma visibly
